  <action>
    Create Next.js app in packages/frontend using the Shadcn preset:

    Run from project root:
    ```bash
    cd packages/frontend
    pnpm create next-app@latest . --typescript --tailwind --eslint --app --src-dir --import-alias "@/*" --use-pnpm
    ```

    If prompted, accept defaults. This creates:
    - Next.js 15 with App Router
    - TypeScript configuration
    - Tailwind CSS
    - ESLint
    - src/ directory structure

    Modify tsconfig.json to extend root:
    - Add: "extends": "../../tsconfig.base.json"
    - Keep Next.js-specific overrides

    Update package.json name to "@lineupiq/frontend"
  </action>
  <verify>cd packages/frontend && pnpm build</verify>
  <done>Next.js app builds successfully</done>
